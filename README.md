##### Instalasi
``` bash
 $ pkg update && pkg upgrade
 $ pkg install python git
 $ pip install requests bs4
 $ git clone https://github.com/AngCyber/Dump
 $ cd $HOME/Dump
 $ python Dump.py
```
##### Update Tools
``` Bash
 $ cd $HOME/Dump
 $ git pull
 $ python Dump.py
```
